Solves the power flow equations using the Newton-Raphson iterative method.

This function updates the bus voltages in the BusData structure using the Newton-Raphson algorithm.

Parameters
YThe bus admittance matrix ($$ Y_{bus} $$).
busDataThe bus data, including initial guesses for voltages and specified powers.
maxIterMaximum number of iterations (default: 1024).
toleranceConvergence tolerance for power mismatches (default: $$ 1 \times 10^{-8} $$).
Returns
true if the algorithm converged within the specified number of iterations, false otherwise.
Note
The function modifies the voltages in the BusData structure in-place.
